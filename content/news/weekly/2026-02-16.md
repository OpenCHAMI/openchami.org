# OpenCHAMI Weekly Digest: Activity Highlights (Feb 06–Feb 13, 2026)

## Highlights
- **Enhanced DNS Security:** A new pull request adds support for the [RRL (Response Rate Limiting) plugin](https://github.com/OpenCHAMI/coresmd/pull/45) to the `coresmd` service, helping mitigate DNS abuse and improve reliability.
- **Flexible DNS Configuration:** Proposed feature to enable [per-interface DNS entries](https://github.com/OpenCHAMI/coresmd/issues/43) for more granular network setups.
- **Improved Metadata Management:** Introduction of [profile-based controls for metadata and cloud-config](https://github.com/OpenCHAMI/metadata-service/issues/5) with a matching pull request adding a new Profile resource and related operations for streamlined configuration.
- **JWT Support Incoming:** An open feature request to [add JWT support in the boot-service client](https://github.com/OpenCHAMI/boot-service/issues/4) aims to enhance secure authentication options.
- **Group Interaction Features for SMD:** Discussion underway for adding [SMD group interaction capabilities](https://github.com/OpenCHAMI/ochami/issues/16) to facilitate better cluster management.
- **Bug Fix Underway:** Investigation into an issue where [sqlite fails on server startup with ent backend](https://github.com/OpenCHAMI/fabrica/issues/28) improves system stability.
- **Remote Console Update:** Refactoring efforts to [expose a websocket interface](https://github.com/OpenCHAMI/remote-console/pull/15) for more interactive remote management.
- **Community and Governance Updates:** Merged pull request updating [TSC member details and replacements](https://github.com/OpenCHAMI/community/pull/40) ensures current project leadership information is accurate.
- **Documentation Enhancements:** Ongoing work to [add Slurm installation docs](https://github.com/OpenCHAMI/openchami.org/pull/72) and reorganize existing content for easier navigation.
- **FRU Tracker Upgrade:** Merged changes to update the [FRU tracking service](https://github.com/OpenCHAMI/fru-tracker/pull/4) to leverage the latest Fabrica improvements.
