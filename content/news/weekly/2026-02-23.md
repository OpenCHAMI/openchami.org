# OpenCHAMI Weekly Digest  
*Feb 09 – Feb 16, 2026*

---

## Highlights

- **Enhanced DNS Security**: The `coresmd` repo added the *Response Rate Limiting (RRL)* plugin, improving DNS server resilience against abuse and attacks. [#45](https://github.com/OpenCHAMI/coresmd/pull/45)  
- **Profile-Based Configuration Control**: In `metadata-service`, new features to manage metadata and cloud-config via profiles enhance flexibility for user environments. [Issue #5](https://github.com/OpenCHAMI/metadata-service/issues/5), [PR #6](https://github.com/OpenCHAMI/metadata-service/pull/6)  
- **FRU Tracker v0.1.0 Released**: The first official release enables tracking of Field Replaceable Units, backed by new functionality tied to Fabrica releases. [Release details](https://github.com/OpenCHAMI/fru-tracker/releases/tag/v0.1.0), [PR #6](https://github.com/OpenCHAMI/fru-tracker/pull/6)  
- **Updated Event Agenda**: The `openchami.org` site expanded the UCL 2026 event agenda with detailed themes, making it easier for users to plan attendance. [#75](https://github.com/OpenCHAMI/openchami.org/pull/75)  
- **Improved Database Connectivity Defaults**: `smd` now supports a default database port, simplifying setup for new users. [#85](https://github.com/OpenCHAMI/smd/pull/85)  
- **Expanded Power Control Support**: `power-control` gained support for JAWS PDU devices, broadening hardware compatibility. [#45](https://github.com/OpenCHAMI/power-control/pull/45)  
- **Documentation Enhancements**: New “Install Slurm” guide published on `openchami.org` to aid users in cluster management installation. [#72](https://github.com/OpenCHAMI/openchami.org/pull/72)  
- **Repository Configuration Flexibility**: `image-builder` added capabilities to write repository configs, offering more control over image generation. [#56](https://github.com/OpenCHAMI/image-builder/pull/56)  
- **Codebase Cleanups and Dependency Updates**: Multiple repos saw code refactoring and dependency bumps for stability and maintainability, including `power-control` and `ochami-rs`.  
- **Weekly Digest Published**: The official OpenCHAMI weekly update was posted for community transparency and awareness. [#74](https://github.com/OpenCHAMI/openchami.org/pull/74)
