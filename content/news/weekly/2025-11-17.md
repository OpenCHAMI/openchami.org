*Nov 10–Nov 17, 2025*
---
## Highlights
- **Documentation polish:** A bug causing long empty space on the [OpenCHAMI tutorial page](https://github.com/OpenCHAMI/openchami.org/issues/50) was reported, aiming to improve the user reading experience.
- **Image build fix in progress:** Users on x86 nodes face building issues with `image-build:v0.1.2` ([issue #50](https://github.com/OpenCHAMI/image-builder/issues/50)), targeting smoother infrastructure setup.
- **Firmware Management Service proposal:** Functional specs for a new service to streamline firmware workflows were initiated ([roadmap #119](https://github.com/OpenCHAMI/roadmap/issues/119)).
- **Event updates:** Multiple conference entries for 2026 added along with refreshed logos on the website, helping users stay informed about upcoming community events ([PR #51](https://github.com/OpenCHAMI/openchami.org/pull/51)).
- **Community growth:** Weekly update PR in community repo shows ongoing engagement to enhance collaboration ([PR #38](https://github.com/OpenCHAMI/community/pull/38)).
- **Ex-bootstrap improvements:** Default file parameter removed to reduce confusion; error messages refined for better clarity ([PR #1](https://github.com/OpenCHAMI/ex-bootstrap/pull/1)).
- **Testing enhancements:** Added tests for a complete example workflow to increase reliability in Fabrica ([PR #23](https://github.com/OpenCHAMI/fabrica/pull/23)).
- **New release:** `ex-bootstrap` published version [v1.0.2](https://github.com/OpenCHAMI/ex-bootstrap/releases/tag/v1.0.2) improving stability and usability.
