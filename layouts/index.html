Security

HPC jobs run in the context of a POSIX filesystem. Cloud APIs depend on access tokens that are limited in both scope and time. OpenCHAMI ships everything you need to bridge the two worlds.

Simplicity

OpenCHAMI can support the largest HPC systems with the most complex requirements, but the default install is useful within five minutes. Complexity is always "opt-in".

Modularity

Each of the components that comprise the OpenCHAMI system is designed to work in a standalone mode and can be fully replaced.
