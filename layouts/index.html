Community

OpenCHAMI is supported by an international community of some of the largest HPC sites in the world as well as some of the smallest.

Simplicity

OpenCHAMI can support the largest HPC systems with the most complex requirements, but the default install can be running in five minutes on a raspberry pi. Administrators get simplicity by design and can opt-in to complexity if they choose.

Modularity

Each of the components that comprise and OpenCHAMI system is designed to work in a standalone mode and can be fully replaced.

Security

HPC jobs run in the context of a POSIX filesystem. Cloud APIs depend on access tokens that are limited in both scope and time. OpenCHAMI ships everything you need to bridge the two worlds.
